        // GroupDocs.Merger APIን በመጠቀም የPDF ፋይል ገጾችን አሽከርክር

        // የውጤት ሰነዱን ለማስቀመጥ የመቆያ ዘዴ ይደውሉ እና የተፈለገውን የፋይል መንገድ ይለፉ
        merger.save("output.pdf");
